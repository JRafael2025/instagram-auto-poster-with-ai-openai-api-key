# LinkedIn Auto Post (APIs Linkedin + OpenAI)

This project is designed to automate posting on LinkedIn using APIs from LinkedIn and OpenAI. It includes token management, folder reading, and integration with the OpenAI platform.

## Purpose and Main Features

The main purpose of this project is to simplify the process of posting updates on LinkedIn by automating the posting process. The key features include token management, reading folders for content, and leveraging OpenAI's capabilities for generating content.

## File Structure Explanation

The main file included in this project is `LinkedIn Auto Post (APIs Linkedin + OpenAI)_.ipynb`, which contains the code for the automation process. It includes cells for imports and configuration, as well as functions for handling the API requests and integrating with OpenAI.

## Key Functions/Classes

1. `Imports and Configuration`: This cell includes the necessary imports and configuration settings for the project, such as API keys, token metadata file, and initializing the OpenAI client.
2. `Token Management`: This cell manages the authentication tokens for LinkedIn and OpenAI.
3. `Folder Reading`: This cell reads folders for content to be posted on LinkedIn.
4. `OpenAI Integration`: This cell integrates with OpenAI for generating content to be posted.

## Usage Instructions

1. Open the `LinkedIn Auto Post (APIs Linkedin + OpenAI)_.ipynb` file in a Jupyter notebook environment.
2. Replace the placeholder values for API keys, tokens, client ID, and client secret with your actual credentials.
3. Run the cells in order to configure the project and start automating the posting process on LinkedIn.

By following these steps, you can effectively automate the posting process on LinkedIn using APIs from LinkedIn and OpenAI.
